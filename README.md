## ISOPACK

#### Installation
```bash
npm i isopack --save-dev
# or
yarn add isopack --dev
```

Copy [iospack.yml.example](https://github.com/mrsum/isopack/blob/master/isopack.yml.example) to isopack.yml


#### package.json
```json
...
"scripts": {
  "dev": "isopack dev",
  "build": "isopack prod"
}
...

```

#### Development
```
npm run dev
```g

#### Building
```
npm run build
```
